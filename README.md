A cutting-edge **3D automotive showroom** built for Schaeffler — one of the world's leading automotive component manufacturers. The app lets users explore **23 individual components** of a 2-wheeler through an immersive, fully interactive 3D environment.

**How it works:**
1. A high-resolution **3D bike model** loads in the browser
2. Users hover over the bike OR the sidebar — the corresponding parts **highlight in real-time**
3. Clicking any component **zooms into an isolated 3D view** of that part
4. Each part shows **technical specifications, features, and advantages**
5. The entire app packages as a **standalone desktop application (Electron)** for trade shows and kiosks

**Technical Highlights:**
- 🎮 Custom **mesh traversal and highlighting** system — O(1) performance optimized
- 🔄 **Zustand state bridge** connecting 3D canvas to HTML UI
- 🖼️ **GLTF model loading** with fuzzy mesh-name matching
- 💻 **Cross-platform** — runs as web app AND packaged desktop app
- ⚡ **60 FPS maintained** through smart material cloning and emissive optimization
- 🎨 **GSAP animations** for smooth transitions

**Business Impact:**  
- Used for **trade shows, client presentations, and kiosk displays**
- Transforms a technical product catalog into an **engaging interactive experience**
- No special hardware needed — runs in any modern browser or as a desktop app
- Dramatically increases **client engagement and product understanding**
